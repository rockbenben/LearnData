# 创意，是一笔灵魂交易

《[创意，是一笔灵魂交易](https://book.douban.com/subject/11517171/)》没读全文，直接使用提出的两种关键实践方法，“晨间笔记”和“艺术家之约”： 这是书中被比作无线电的接收器和发射台。

- 晨间笔记： 每天早上写三页手稿，是一种意识流的记录，类似“大脑排污管”，旨在清空思绪，区分真假，帮助放下评判，连接直觉。无论内容多么琐碎、负面或愚蠢，都必须坚持书写，并且**在前八周内不进行回顾**。它帮助人们认识到情绪不重要，重要的只是书写行为本身。
  - 引用： “简单地说，晨间笔记就是三页手稿。更严格点儿说，它是一种意识流：‘哦，天哪，又到早上了。新的一天开始了，我却没有一丁点儿值得说的东西。我得洗窗帘，对了，昨天我那些脏衣服洗了吗？……’虽然将它称为‘大脑排污管’听上去不是那么文雅，但是那的确是它的一项主要功能。”
  - 引用： “不要犹豫，把脑中闪现的任何想法都写进这三页纸之中，就是这样。如果你想不出有什么要写的，就可以写：‘我想不出任何要写的东西……’ 一直写下去，直到写满三页纸。”
  - 引用： “晨间笔记对每个人都奏效，无论你是画家、雕塑家、诗人、演员、律师，还是家庭主妇。”
  - 如果真觉得3页很难，可以将字写得大些。
  - 晨间笔记是“大脑排污管”，要将其与笔谈分开进行。前期不反复阅读晨间笔记，尤其是前8周不翻回来读，是为了防止你自己的审判官立即跳出来批评这些原始的、未经过滤的思绪。由于这些笔记是“大脑排污管”里倾倒出的任何东西，不受评判标准约束，如果立即去审视它们，很容易引发自我怀疑和批评，而这正是晨间笔记练习想要绕过和化解的障碍。将它们藏起来或放在大牛皮纸袋中也进一步强化了这种不对其进行即时评判的理念。
- 艺术家之约： 每周约两小时，专门用于滋养创造意识和内心的“创意小孩”。通常是单独外出远足，不受任何人干扰，目的是为了接受顿悟、灵感和指引。它被强调是绝对必要的，即使觉得荒谬或没有时间，这也说明存在抗拒。这是一种补充“意象之鱼”以“蓄满水池”的手段。
  - 引用： “艺术家之约是每周腾出两小时，专门用来滋养你的创造意识和内心的艺术家。”
  - 引用： “赴艺术家之约则是接受，是向顿悟、灵感和指引敞开心扉。”
  - 引用： “因排不开时间而与艺术家之约失之交臂，付出的代价太大了。”
